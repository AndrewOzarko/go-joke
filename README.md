## Installation && Usage

- `git clone https://github.com/AndrewOzarko/go-joke`
- `cd go-joke`
- `go run user/main.go`
- `open your browser http://localhost:8082`

## Or by docker

- `git clone https://github.com/AndrewOzarko/go-joke`
- `cd go-joke`
- `docker-compose build`
- `docker-compose up -d`
- `open your browser http://localhost:8082`
